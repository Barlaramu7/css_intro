HTML Tags with Descriptions

html
This is the root tag of every HTML page.
head
It contains page settings and links to CSS or JS files.
title
Sets the title of the web page (shown in browser tab).
body
This contains everything shown on the webpage.
h1 to h6
Headings from largest (h1) to smallest (h6).
p
This tag adds a paragraph of text.
a
Used to add a hyperlink (link to another page).
img
Used to display images.
ul, ol, li
Used for lists (unordered, ordered, and list items).
div
A container used to group elements.
span
Inline container for styling small text.
